## Getting Started

### As a Git Submodule

> **Note:** Before proceeding, ensure you have [Git](https://git-scm.com) and [Hugo](https://gohugo.io) installed and that you have initialized a new Hugo project.

From the root of your Hugo project, initialize Git:

```shell
git init
```

Then add Kansō as a submodule:

```shell
git submodule add https://github.com/bitgnostiq/kanso themes/kanso
```
